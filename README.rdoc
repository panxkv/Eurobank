== README
https://eurobank.herokuapp.com/

Aplikacaj do wspierania procesu rekrutacji (zgłoszenie kandydatury na stanowisko, przyjęcie/ odrzucenie aplikacji).

Specyfikacja:

* Ruby 2.3.0

* Rails 4.2.0

* SQLite

* Bootstrap

Uruchom:

<tt>gem install bundle</tt>
<tt>bundle install</tt>
<tt>rake db:create</tt>
<tt>rake db:migrate</tt>
<tt>rake db:seed</tt>
